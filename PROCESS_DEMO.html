Step 2 · Transcriber

URL 하나 → 스크립트 + Notion 저장까지 원클릭

Streamlit 웹앱. 영상 URL 을 붙여넣고 실행 버튼만 누르면 4단계가 순서대로 돌아갑니다.

localhost:8501 — Streamlit
🎬 Meta 광고 → Notion 트랜스크립터
Meta CDN 또는 Ad Library URL을 붙여넣으면 Whisper로 스크립트를 뽑아 Notion DB에 저장합니다.
⏳ 영상 다운로드

⏳ Notion 적재

Under The Hood

각 단계가 실제로 하는 일

네 개의 모듈이 각자의 일만 담당하도록 분리. 한 단계가 실패해도 어디서 끊겼는지 바로 보입니다.

01

영상 다운로드

URL 타입을 자동 판별. fbcdn.net/cdninstagram.com 이면 requests 로 직접 스트리밍, facebook.com/ads/library 면 yt-dlp 로 우회. 403/404/410 은 "URL 만료" 로 친절하게 안내.

02

오디오 추출

ffmpeg 으로 mono 16kHz mp3 추출. 64kbps 로 시작해서 Whisper 25MB 제한 넘으면 48→32→24→16kbps 로 자동 낮춤. Windows 한글 경로 cp949 크래시 방지 처리 포함.

03

Whisper 스크립트

OpenAI whisper-1 모델에 language=ko 로 호출. 인증/레이트리밋/사이즈초과/네트워크 에러를 각각 한국어 메시지로 분류해서 표시.

04

Notion 적재

2025-09+ Notion API 의 data_source 모델 지원. DB ID 를 넣어도, data_source ID 를 넣어도 알아서 resolve. 2000자 초과 스크립트는 chunk 로 쪼개서 rich_text 에 저장 + 페이지 본문에도 전문 백업.
